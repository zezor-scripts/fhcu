Scripting Language Syntax Cheat Sheet

Fundamentals for Web, Game, and System Development

JavaScript (JS) - Frontend & Backend

The core language for web interactivity. Runs in browsers (Frontend) and on servers (Node.js/Backend).

Example 1: Defining a Function

const message = "Hello, World!"; // Variable declaration
function calculateArea(length, width) {
  return length * width;
}
console.log(calculateArea(10, 5));

Example 2: DOM Interaction (Button Click)

// Get an HTML element by its ID
const button = document.getElementById('myButton');
const textDiv = document.getElementById('output');

// Add an event listener to run a function when clicked
button.addEventListener('click', (event) => {
  textDiv.textContent = "Button was clicked!";
  button.style.backgroundColor = 'red';
});
Python - General Purpose & Automation

Known for its clean, readable syntax and extensive libraries. Widely used for data, AI, and scripting.

Example 1: List Iteration and Conditionals

fruits = ['apple', 'banana', 'cherry']
for fruit in fruits:
    if fruit == 'banana':
        print("Found the banana!")
    else:
        print(fruit)

Example 2: File Reading and Writing

# Writing content to a file
with open('log.txt', 'w') as file:
    file.write("Application started at 12:00\n")

# Reading content from a file
with open('log.txt', 'r') as file:
    content = file.read()
    print(content)
Lua - Lightweight & Embedded Systems (Roblox)

A fast, small, and embeddable scripting language. Famous as the core scripting language for Roblox Studio.

Example 1: Tables (Dictionaries/Arrays)

local playerStats = {
  Name = "Zezor", -- Key-value pairs (dictionary)
  XP = 500,
  "sword", "shield" -- Array elements
}
print(playerStats.XP) -- Access the value 500
print(playerStats[2]) -- Access the array element "shield"

Example 2: Handling a Roblox Event

local part = game.Workspace.TeleportPad

local function onTouched(otherPart)
  -- Find the player's character model
  local character = otherPart.Parent
  local humanoid = character:FindFirstChild("Humanoid")

  if humanoid then
    -- Teleport the character to a new location (Vector3)
    character.PrimaryPart.CFrame = CFrame.new(0, 100, 0)
  end
end

part.Touched:Connect(onTouched) -- Connect the function to the event
SQL - Database Management

Structured Query Language. Used to communicate with databases to store, retrieve, and manage data.

Example 1: Basic Data Selection

SELECT name, score -- Select the columns you want
FROM high_scores -- Specify the table
WHERE score > 1000 -- Filter the results
ORDER BY score DESC; -- Sort the results (Descending)

Example 2: Joining Tables and Aggregation

SELECT p.project_name, COUNT(t.task_id) AS total_tasks
FROM projects p
JOIN tasks t ON p.project_id = t.project_id -- Connects two tables
WHERE t.status = 'In Progress'
GROUP BY p.project_name
HAVING COUNT(t.task_id) > 5; -- Filters groups
PHP - Server-Side Web Development

A powerful language primarily used for server-side scripting (processing forms, managing databases, generating HTML).

Example 1: Combining Variables and HTML

<?php // PHP code starts here
  $username = "CodeMaster";
  $age = 25;
?>
<h1>Welcome, <?php echo $username; ?>!</h1>
// Output: Welcome, CodeMaster!

Example 2: Handling a Form Submission (POST)

<?php
if ($_SERVER['REQUEST_METHOD'] === 'POST') {
  // Check if the form was submitted
  $user_input = $_POST['name_field'];
  $safe_input = htmlspecialchars($user_input); // Sanitize input!
  echo "Thank you, " . $safe_input;
}
// Note: This requires a form tag in the HTML with method="POST"
?>
Bash / Shell - System Scripting

Used to write scripts that automate tasks on Linux/Unix systems and command lines.

Example 1: Simple Backup Script

#!/bin/bash
echo "Starting backup..."
DEST=/home/user/backups/$(date +%Y-%m-%d)
mkdir -p $DEST # Create backup directory
cp -r ~/Documents/* $DEST
echo "Backup complete to "$DEST

Example 2: Checking Status and Printing System Info

# Define a function
check_dir() {
  if [ -d "$1" ]; then
    echo "Directory $1 exists."
  else
    echo "Error: Directory $1 not found."
    exit 1
  fi
}
# Call the function and get system info
check_dir /var/log
echo "Running processes:"
ps -aux | head -n 5
